# История изменений

## Версия 1.0.3 (Стабильная) - 03.01.2026

### Исправления
- ✅ Исправлена отзывчивость веб-интерфейса (убраны блокирующие операции)
- ✅ Исправлено WiFi сканирование (чтение результатов напрямую из WiFi)
- ✅ Исправлено отображение системной информации
- ✅ Добавлена OTA загрузка через веб-интерфейс
- ✅ Оптимизированы задержки в основном цикле (1мс вместо 10мс)
- ✅ Увеличен интервал обновления данных в вебе (5 сек вместо 2 сек)
- ✅ Добавлено отложенное сохранение в EEPROM (через 2 сек после изменения)
- ✅ Убран блокирующий delay() из обработчиков перезагрузки
- ✅ Добавлен обработчик 404 для несуществующих путей
- ✅ Исправлена потоковая передача HTML файла (не обрезается)

### Улучшения
- ⬆️ MQTT подключение с таймаутом 2 секунды (неблокирующее)
- ⬆️ MQTT публикация без подтверждения (QoS 0) для скорости
- ⬆️ Обновление OLED дисплея 1 раз в секунду (вместо каждые 500мс)
- ⬆️ Переподключение MQTT не чаще 1 раза в 10 секунд

### Новые функции
- ➕ OTA загрузка прошивки через веб-интерфейс
- ➕ Страница "Подключение" с инструкциями по пинам
- ➕ Отображение температуры и WiFi статуса на OLED
- ➕ Управление уставкой через энкодер
- ➕ Подброс угля через энкодер (кнопка) и веб-интерфейс
