# Debug Version - Минимальная версия для отладки

**Версия:** Debug  
**Дата:** 2024-12-XX

## Описание

Минимальная версия прошивки для отладки. Содержит только:
- Чтение и отображение температур с датчиков DS18B20
- Простое управление реле (вентилятор, насос, датчики)
- Минимальный веб-интерфейс на одной странице
- Отображение температуры подачи на OLED дисплее

## Функции

### Отображение температур
- Подача
- Обратка
- Котельная
- Улица

### Управление реле
- Вентилятор (вкл/выкл)
- Насос (вкл/выкл)
- Датчики (вкл/выкл)

### OLED дисплей
- Температура подачи (крупным шрифтом)
- Метка "Подача"
- WiFi статус/IP адрес

## Что НЕ включено

- Логика автоматического управления
- MQTT
- NTP
- OTA обновления
- Энкодер
- Подброс угля
- Все настройки и конфигурации
- EEPROM сохранение

## Использование

1. Скопировать файлы в проект:
   - `main.cpp` → `src/main.cpp`
   - `index.html` → `data/index.html`
   - `platformio.ini` → `platformio.ini`

2. Скомпилировать и загрузить:
   ```bash
   pio run -e esp32dev -t upload
   pio run -e esp32dev -t uploadfs
   ```

## Веб-интерфейс

Откройте в браузере IP адрес ESP32. Интерфейс содержит:
- Отображение всех температур (обновление каждые 2 секунды)
- Кнопки управления реле (вкл/выкл)

## API

- `GET /api/status` - получение статуса (температуры и состояния реле)
- `POST /api/control` - управление реле
  - Параметры: `device` (fan/pump/sensors), `state` (true/false)
