# Быстрый старт

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Запуск приложения

```bash
python run.py
```

## Использование

1. **Подключение к MQTT:**
   - Нажмите кнопку "Подключиться" в верхней панели
   - По умолчанию используются настройки из PROMPT.md
   - Для изменения настроек нажмите "Настройки"

2. **Сохранение данных:**
   - Нажмите "Начать сохранение" для записи данных в файл
   - Файлы сохраняются в папку `data/` в формате JSON Lines
   - Имя файла: `ml_data_YYYY-MM-DD_HH-MM-SS.jsonl`

3. **Просмотр данных:**
   - **Вкладка "Мониторинг"**: Температуры и состояния устройств
   - **Вкладка "Графики"**: Графики температур в реальном времени
   - **Вкладка "Статистика"**: Системная информация и статистика приложения

## Настройки MQTT по умолчанию

- Порт: `5374`
