# Инструкция по загрузке OTA

## Изменения в прошивке:

1. ✅ **Исправлена логика датчика обратки** - теперь принимает значения если изменение < 10°C (более мягкая проверка)
2. ✅ **Увеличена карточка подачи и уставки в 3 раза** - добавлены специальные стили с классами

## Загрузка через VS Code (PlatformIO):

### Вариант 1: Через терминал VS Code
```bash
# Загрузка прошивки
pio run -e esp32dev_ota -t upload

# Загрузка SPIFFS (веб-интерфейс)
pio run -e esp32dev_ota -t uploadfs
```

### Вариант 2: Через кнопки в VS Code
1. Нажмите на кнопку "Upload" в нижней панели PlatformIO
2. Выберите окружение `esp32dev_ota`
3. После загрузки прошивки загрузите SPIFFS через "Upload File System image"

## Файлы для загрузки:

- **Прошивка**: `.pio/build/esp32dev_ota/firmware.bin`
- **SPIFFS**: `.pio/build/esp32dev_ota/spiffs.bin`
