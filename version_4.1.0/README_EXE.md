# ML Training - Исполняемый файл для Windows

## Быстрый старт

### Для пользователей (без Python):

1. **Скачайте файл:** `ML_Training_Kotel.exe`
2. **Запустите:** Двойной клик на файл
3. **Используйте:**
   - Нажмите "Подключиться" для подключения к MQTT
   - Нажмите "Начать сохранение" для записи данных
   - Данные сохраняются в папке `data/` рядом с exe файлом

### Структура файлов:

```
ML_Training_Kotel.exe    # Главный файл приложения
data/                    # Папка для сохранения данных (создается автоматически)
  └── ml_data_*.jsonl   # Файлы с данными
app.log                  # Лог файл (создается автоматически)
```

## Для разработчиков (сборка exe)

См. файл `BUILD_INSTRUCTIONS.md` для инструкций по сборке.

## Системные требования

- **ОС:** Windows 10/11 (64-bit)
- **RAM:** Минимум 2 ГБ
- **Место на диске:** ~100 МБ для приложения + место для данных
- **Интернет:** Требуется для подключения к MQTT брокеру

## Настройки по умолчанию

- **MQTT Брокер:** m5.wqtt.ru:5374
- **Топик:** kotel/device1/ml/data
- **Интервал данных:** 10 секунд (настраивается в ESP32)

## Изменение настроек

1. Запустите приложение
2. Нажмите кнопку "Настройки"
3. Введите новые параметры MQTT
4. Нажмите "Сохранить"

## Сохранение данных

1. Нажмите "Начать сохранение"
2. Данные будут сохраняться в файлы формата JSON Lines
3. Имя файла: `ml_data_YYYY-MM-DD_HH-MM-SS.jsonl`
4. Файлы находятся в папке `data/` рядом с exe

## Решение проблем

### Приложение не запускается

- Убедитесь, что у вас Windows 10/11 (64-bit)
- Проверьте, не блокирует ли антивирус файл
- Попробуйте запустить от имени администратора

### Не подключается к MQTT

- Проверьте интернет соединение
- Убедитесь, что ESP32 публикует данные
- Проверьте настройки MQTT в приложении

### Данные не сохраняются

- Убедитесь, что нажата кнопка "Начать сохранение"
- Проверьте права на запись в папку с exe файлом
- Посмотрите файл `app.log` для ошибок

## Контакты и поддержка

При возникновении проблем проверьте:
- Файл `app.log` - содержит подробные логи работы
- Консоль (если запускаете из командной строки)
