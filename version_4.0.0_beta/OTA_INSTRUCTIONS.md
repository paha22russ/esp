# Инструкция по загрузке прошивки и веб-интерфейса по OTA

## Способ 1: Через веб-интерфейс (рекомендуется)

### Шаг 1: Загрузка веб-интерфейса

1. Откройте браузер и перейдите по адресу: **http://192.168.0.198**
2. Откройте меню (☰) → **Система**
3. В разделе **OTA Обновление**:
   - Нажмите **"Выбрать файл"**
   - Выберите файл: `.pio\build\esp32dev\spiffs.bin` (это веб-интерфейс)
   - Нажмите **"Загрузить прошивку"**
   - Дождитесь завершения

### Шаг 2: Загрузка прошивки

1. В том же разделе **OTA Обновление**:
   - Нажмите **"Выбрать файл"**
   - Выберите файл: `.pio\build\esp32dev\firmware.bin` (это прошивка)
   - Нажмите **"Загрузить прошивку"**
   - Дождитесь завершения (устройство перезагрузится)

## Способ 2: Через USB (если веб-интерфейс не работает)

### Загрузка веб-интерфейса:
```bash
pio run -e esp32dev -t uploadfs --upload-port COM3
```

### Загрузка прошивки:
```bash
pio run -e esp32dev -t upload --upload-port COM3
```

## Расположение файлов:

- **Веб-интерфейс (SPIFFS):** `.pio\build\esp32dev\spiffs.bin`
- **Прошивка:** `.pio\build\esp32dev\firmware.bin`

## Примечание:

Если устройство не отвечает, подождите 10-15 секунд после перезагрузки и попробуйте снова.
