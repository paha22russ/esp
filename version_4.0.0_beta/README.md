# Версия 4.0.0 Beta

Стабильная версия прошивки для ESP32 котла.

## Дата создания
Создана из стабильной версии проекта.

## Структура проекта

- `src/` - исходный код прошивки
- `include/` - заголовочные файлы
- `data/` - файлы для SPIFFS (веб-интерфейс)
- `lib/` - библиотеки проекта
- `platformio.ini` - конфигурация PlatformIO

## Основные функции

- Управление котлом (вентилятор, насос)
- Мониторинг температуры (подача, обратка, уличная)
- Веб-интерфейс для управления и мониторинга
- MQTT публикация данных
- OTA обновления
- OLED дисплей
- Энкодер для управления

## Компиляция и загрузка

```bash
# Компиляция
pio run -e esp32dev_ota

# Загрузка прошивки
pio run -e esp32dev_ota -t upload

# Загрузка SPIFFS
pio run -e esp32dev_ota -t uploadfs
```

## OTA обновление

Используйте скрипты из корневой папки проекта или PlatformIO в VS Code/Cursor.
