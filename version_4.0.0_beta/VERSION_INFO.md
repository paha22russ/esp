# Информация о версии 4.0.0 Beta

## Дата создания
Версия создана из стабильной версии проекта.

## Статус
Beta - стабильная версия для тестирования

## Что включено

### Исходный код
- `src/main.cpp` - основной файл прошивки
- `src/html_page.txt` - шаблон HTML страницы
- `include/` - все заголовочные файлы проекта

### Веб-интерфейс
- `data/index.html` - веб-интерфейс для управления котлом

### Конфигурация
- `platformio.ini` - конфигурация PlatformIO для сборки проекта

### Документация
- `README.md` - основная документация
- `README_PROJECT.md` - описание проекта
- `README_WEB_INTERFACE.md` - описание веб-интерфейса
- `CHANGELOG_v3.0.6.md` - история изменений
- `OTA_INSTRUCTIONS.md` - инструкции по OTA обновлению
- `CURSOR_SETUP.md` - настройка Cursor IDE

### Скрипты
- Все Python скрипты для OTA загрузки
- Скрипты для сборки и загрузки

## Использование

Эта версия является стабильной и готова к использованию. 
Все функции протестированы и работают корректно.
